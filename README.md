# Chroma Drencher

Animated background made in C with SDL 2 and OpenGL 4.5, based on the original [JavaScript version](https://github.com/lanakhive/chromadrencher).

*Watch color calmly drip down your desktop.*

## Controls

| Key | Action             |
|-----|--------------------|
| F   | Toggle Full Screen |
| S   | Toggle VSync       |
| P   | Pause              |
| Esc | Quit               |
